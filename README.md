# D Mod Manager

Утилита для автоматической установки, обновления и дополнительной настройки модпака для игры **«Мир Танков»**.

Программа позволяет автоматически отслеживать выход новых версий модпака, скачивать обновления, устанавливать дополнительные пользовательские моды и применять собственные настройки после каждого обновления.

---

# Установка

Для работы программы необходимо установить **AutoHotkey v1**.

Скачать AutoHotkey можно с официального сайта:

https://www.autohotkey.com

После установки AutoHotkey просто запустите файл:

```text
Menu.ahk
```

Никаких дополнительных действий для запуска не требуется.

---

# Сохранение настроек при обновлении программы

При переходе на новую версию программы вы можете сохранить все свои настройки.

Для этого скопируйте файл:

```text
libs\SavedSettings.ini
```

из папки старой версии программы в аналогичную папку новой версии.

После запуска новой версии все сохранённые настройки будут автоматически восстановлены.

Это избавляет от необходимости заново настраивать программу после каждого обновления.

---

# Установка дополнительных модов

Если вы хотите использовать собственные моды помимо стандартного набора, поместите их в папку:

```text
Additional mods
```

Во время установки или обновления модпака программа автоматически обработает содержимое этой папки и установит ваши дополнительные моды вместе с основным комплектом.

Эта возможность особенно полезна, если вы используете собственные прицелы, интерфейсные модификации, звуковые пакеты или любые другие дополнения, которые отсутствуют в основном модпаке.

# Автоматическое редактирование конфигов

После установки модов программа может автоматически изменять параметры в конфигурационных файлах.

Настройки находятся в файле:

```text
ConfigEditor.ahk
```

Данная функция позволяет автоматически применять ваши персональные настройки после каждого обновления модпака и избавляет от необходимости вручную редактировать конфиги снова и снова.

Пример настройки:

```ahk
fConfigEditor("ПутьКФайлу"
    , "Что ищем", "На что заменяем"
    , "Что ищем 2", "На что заменяем 2")
```

Принцип работы:

* Первый параметр — путь к конфигурационному файлу.
* Далее параметры передаются парами.
* Первый элемент пары — текст для поиска.
* Второй элемент пары — текст для замены.
* Количество замен для одного файла не ограничено.
* Количество обрабатываемых файлов также не ограничено.

При каждом обновлении программа будет автоматически вносить указанные изменения в выбранные конфигурационные файлы.

Дополнительно доступен параметр:

```ahk
copyErrorReport := False
```

Если установить значение `True`, программа будет формировать отчёт об ошибках копирования файлов, что может помочь при диагностике проблем.

# Дополнительные настройки

Расширенные параметры находятся в файле:

```text
AdvancedSettings.ahk
```

Большинству пользователей изменять эти настройки не требуется.

В данном файле находятся различные служебные параметры программы, включая таймауты ожидания, ограничения повторных попыток, настройки поиска изображений и другие внутренние параметры.

Если программа работает нестабильно на вашем компьютере или возникают проблемы при установке и обновлении модов, можно попробовать изменить некоторые значения в этом файле.

Перед внесением изменений рекомендуется сохранить резервную копию.

# Экстренные горячие клавиши

Для аварийного управления программой предусмотрены специальные горячие клавиши.

По умолчанию используются:

| Клавиша | Действие                    |
| ------- | --------------------------- |
| Insert  | Перезапуск программы        |
| Delete  | Полное завершение программы |

Если программа зависла или возникла непредвиденная ситуация, эти клавиши позволяют быстро перезапустить или остановить скрипт.

При необходимости их можно изменить в файле:

```text
AdvancedSettings.ahk
```

Параметры:

```ahk
ReloadScriptKey
StopScriptKey
```

# AutopilotEBR

В основной папке программы также находится файл:

```text
AutopilotEBR.ahk
```

Это экспериментальный мод для автоматического переключения режимов колесной техники.

⚠️ В настоящее время мод находится в стадии тестирования и может работать нестабильно. Возможны ошибки и некорректная работа в отдельных игровых ситуациях.

Используйте данную функцию на свой страх и риск.
